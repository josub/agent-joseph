existing memory already answered. He called it out: "this question is you not doing your
research" and "you need to catch up."

material. As his co-founder I'm expected to come up to speed myself.

**How to apply:** Before asking, exhaust the obvious sources: repo docs (README, ROADMAP,
PROGRESS, docs/), this brain's memory (recall first), and git history. Only ask when the
